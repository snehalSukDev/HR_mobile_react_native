# HR Mobile React Native

A React Native mobile app for Frappe HRMS built with Expo SDK 54.

## Getting Started
- Install dependencies: `npm install`
- Start development: `npx expo start`
- Scan the QR in Expo Go to run on device

## Build
- Android APK: `npx eas build -p android --profile preview`
- Android AAB: `npx eas build -p android --profile production`
